# homebrew-tap

Homebrew tap for [warden](https://github.com/srjn45/warden) — run a fleet of
coding agents without losing your mind.

```sh
brew install --cask srjn45/tap/warden
```

The cask under `Casks/` is generated and pushed automatically by the
[warden release pipeline](https://github.com/srjn45/warden/blob/main/.goreleaser.yaml)
on every tagged release — don't edit it by hand.
